📖 Universal Template Engine Guide

🔧 Managing Templates

  1. Access Manager: Enter the password to unlock the Template Manager.
  2. Create & Edit: Fill out the form to create or edit a template.
  3. Save Template: Click "Save Template" to save your work to the current session. You can save multiple times.
  4. Download File: When you are finished, click "Download Standalone File" to save a new HTML file with all your changes.
  5. Use Anywhere: The downloaded file is now your new, standalone application.

📝 Generating Messages

  1. Select Category & Case: Choose your desired template from the dropdowns.
  2. Fill Variables: Complete the input fields that appear.
  3. Generate & Copy: Create your message and copy it to the clipboard.

🔧 إدارة القوالب

  1. الوصول للمدير: أدخل كلمة المرور لإلغاء قفل مدير القوالب.
  2. إنشاء وتعديل: املأ النموذج لإنشاء أو تعديل قالب.
  3. حفظ القالب: انقر على "حفظ القالب" لحفظ عملك في الجلسة الحالية. يمكنك الحفظ عدة مرات.
